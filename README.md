# AI-Pengfei-Solo-Campus-Brand
AI低碳能源科普内容生成与校园传播风控系统｜AI先锋未来人才大赛｜鹏飞集团真实命题
## 项目背景
鹏飞集团正由传统能源企业转型绿色低碳氢能科技企业，目前在校招校园传播存在三大痛点：
1. 品牌刻板印象：学生仍将鹏飞认知为重化工传统企业，不了解氢能全产业链、零碳智慧物流等新兴低碳业务
2. 内容年轻化不足：官方宣讲素材形式单一，不符合B站、小红书、高校社群年轻群体传播习惯
3. 精准触达不足：不同高校、专业学生信息渠道差异大，难以做到分层精准传播
## 解决方案
本单人参赛项目包含三大核心模块：
1. AI低碳能源科普内容生成模块
    - 通过大模型提示词模板，批量生成B站短视频脚本、小红书科普笔记、高校社群校招文案
    - 将氢能、零碳技术专业知识转化为轻量化年轻化内容，破除刻板印象
2. SQL高校学生画像数据分析模块
    - 通过调研问卷数据，使用SQL完成高校学生专业、年级、信息渠道、能源认知画像分析
    - 输出精准分层校园传播策略，实现差异化内容投放
3. 能源宣传合规风控模块
    - 建立广告宣传合规模板、学生隐私合规规范，保障校园推广内容合规安全
## 技术栈
- SQL：校园用户画像数据分析
- 大模型API：AI科普文案生成
- Markdown/PPT：方案文档 & 答辩材料
- 剪映：3分钟项目演示视频
## 成果
- 全套氢能科普校园文案模板
- SQL数据分析脚本与可视化报告
- 完整鹏飞集团校园雇主品牌传播策划方案
- 3分钟项目演示视频
- 能源宣传合规审核文档
